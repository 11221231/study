1. 기본 공식 및 운동방정식

  • $F = ma = mg = kx$
  • 근의 공식: $x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$
  • 오일러 공식: $e^{\theta i} = \cos\theta + i\sin\theta$

방정식의 해를 구하는 방법

  • 정해: 선형, 탄성 계 $\Rightarrow$ 시간에 따른 $t$ 값을 대입해서 직접 풀어지는 것
  • 수치해: 비선형, 비탄성 계 $\Rightarrow$ $\Delta t$ (짧은 시간) 간격을 두고 이전 상태를 이용해 풀어가는 것

기본 운동방정식 형태

$$mu'' + cu' + ku = f$$
  • $mu''$ : 관성력 (질량 $\times$ 가속도)
  • $cu'$ : 감쇠력 (감쇠 $\times$ 속도)
  • $ku$ : 복원력 (강성 $\times$ 변위)

전체 동적응답($u_d$)은 외력이 없는 자유진동($u_c$)과 외력에 의한 강제진동($u_p$)의 합으로 구성됩니다. ($u_d = u_c + u_p$)

2. 비감쇠 시스템 (감쇠 X)

운동방정식: $mu'' + ku = f$

① 자유진동 풀이 ($f=0$)

$u_c = e^{at}$ 로 가정 후 미분하여 $m(a^2e^{at}) + k(e^{at}) = 0$ 도출.

특성방정식 $ma^2 + k = 0$ 에서 $a = \pm \sqrt{\frac{k}{m}}i = \pm \omega_n i$ 를 얻습니다.

고유진동수: $\omega_n = \sqrt{\frac{k}{m}}$

최종 해 (오일러 공식 적용 및 초기조건 대입):
$$u(t) = u(0)\cos\omega_nt + \frac{u'(0)}{\omega_n}\sin\omega_nt$$

② 강제진동 풀이 ($f \neq 0$)

$mu_p'' + ku_p = F$ ($F$를 상수하중이라 가정 시 정적응답 $u_p = \frac{F}{k}$)

최종 동적응답 (초기 정지 상태 가정):
$$u_d(t) = \frac{F}{k}(1 - \cos\omega_nt)$$
고유주기: $T_n = \frac{2\pi}{\omega_n} = 2\pi\sqrt{\frac{m}{k}}$

3. 감쇠 시스템 (감쇠 O)

  • 점성감쇠 (Viscous): 속도에 비례하여 감소.
  • 마찰감쇠 (Friction): 속도의 반대방향으로 일정하게 작용하는 비선형성 감쇠.

점성감쇠 운동방정식: $mu'' + cu' + ku = 0$

특성방정식 $ma^2 + ca + k = 0$ 의 근의 공식을 통해 임계감쇠를 구합니다.

감쇠비($\xi$)와 임계감쇠($c_c$)

  • 임계감쇠: 판별식 $c^2 - 4mk = 0$ 일 때, $c_c = 2\sqrt{mk} = 2m\omega_n$
  • 감쇠비: $\xi = \frac{c}{c_c}$ $\Rightarrow$ $c = 2\xi\omega_n m$

부족감쇠 해 ($c^2 - 4mk < 0$)

감쇠고유진동수 $\omega_d = \omega_n\sqrt{1-\xi^2}$

$$u(t) = e^{-\xi\omega_nt}(A\cos\omega_dt + B\sin\omega_dt)$$

※ $e^{-\xi\omega_nt}$ 형태의 감소하는 지수함수 곡선 안에서 주기운동을 하며 수렴합니다.

4. 감쇠비와 대수감쇠율 (진폭 감소율)

감쇠비 $\xi$가 5% 이하일 때는 $\omega_n \approx \omega_d$ 로 근사치를 사용합니다.

주기에 따른 진폭비 ($j$회 진동 시):
$$\frac{u_{i+j}}{u_i} \approx e^{-2\pi\xi j}$$

예제: 진폭이 50% 감소하는 데 걸리는 진동 횟수($j$)

$0.5 = e^{-2\pi\xi j}$ 양변에 자연로그를 취하면, $\ln(0.5) = -2\pi\xi j$

$j = \frac{\ln(0.5)}{-2\pi\xi} \approx \frac{0.11}{\xi}$

만약 건물의 감쇠비가 2% ($\xi = 0.02$) 라면, $j = \frac{0.11}{0.02} = 5.5$

$\Rightarrow$ 즉, 5.5회 진동하면 건물의 진폭이 반으로 줄어듭니다.

5. [시뮬레이션] 점성감쇠 vs 마찰감쇠 그래프

점성감쇠는 진폭이 지수함수적으로 감소하지만, 쿨롱 마찰에 의한 마찰감쇠는 한 주기마다 $\frac{4F}{k}$ 씩 선형적(직선)으로 감소하여 최종적으로 멈추게 됩니다.

(기본 조건: 질량 $m=1\text{kg}$, 강성 $k=40\text{N/m}$, 초기변위 $u_0=1\text{m}$, 점성감쇠비 $\xi=0.04$)
