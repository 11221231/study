Part 1

내진설계 이론 및 원리편

여기서는 “공식을 외우기 전에 왜 그런 모양이 나오는지”를 먼저 이해합니다. 기준식은 그대로 가져오되, 말투는 최대한 부드럽게 풀었습니다.

1. 건축물 주기 \(T\)의 역학적 이해

주기는 내진설계의 출발점입니다. 건물이 얼마나 빠르게 흔들리는지를 뜻하고, 이 값이 응답스펙트럼 위에서 어디에 놓이느냐에 따라 설계지진력이 달라집니다.

1-1. 고유주기란?

T = 2π √(M / K)
T : 고유주기(초). 건물이 한 번 흔들리고 원래 상태로 돌아오는 데 필요한 대표 시간입니다.
M : 유효질량. 건물이 “얼마나 무겁게” 흔들리는지의 기준입니다.
K : 유효강성. 건물이 옆으로 밀렸을 때 얼마나 “버티려는지”를 나타냅니다.

이 식을 아주 쉽게 읽으면 이렇습니다. 무거우면 느려지고, 뻣뻣하면 빨라집니다. 그래서 높은 건물이 대체로 주기가 길고, 낮고 단단한 건물은 주기가 짧습니다.

F = M · a

지진력은 결국 관성력입니다. 같은 질량이라도 응답가속도 a가 커지면 힘이 커집니다. 그래서 KDS의 응답스펙트럼은 “주기에 따라 응답가속도 \(S_a\)가 어떻게 달라지나”를 보여주는 그래프입니다.

짧은 주기 vs 긴 주기, 느낌으로 보면

주기가 짧다 = 빠르게 반응 주기가 길다 = 천천히 크게 흔들림

1-2. 응답스펙트럼은 왜 “가속도-속도-변위 민감 구간”으로 나뉠까?

Sᵥ ≈ (T / 2π) · Sₐ
S_d ≈ (T / 2π)² · Sₐ
짧은 주기 구간 \(T \to 0\) 쪽에서는 건물이 거의 강체처럼 거동합니다. 지반 움직임을 빠르게 따라가므로 상대변위는 작아도 절대가속도는 커질 수 있습니다.
중간 주기 구간 여기서는 스펙트럼이 대체로 속도 지배 성격을 띱니다. 그래서 \(T_S\) 이후에 \(1/T\) 형태가 등장합니다.
긴 주기 구간 \(T \to \infty\) 쪽에서는 구조가 매우 유연해집니다. 가속도는 줄어들 수 있지만 변위는 커지기 쉬워집니다.

그래서 응답스펙트럼을 읽을 때는 “이 건물이 지금 힘에 민감한 위치에 있는지, 아니면 변위에 민감한 위치에 있는지”를 같이 보는 습관이 중요합니다.

T ≤ Cᵤ · Tₐ

수치해석 모델이 너무 유연하게 잡히면 주기가 과도하게 길어져 지진력이 과소평가될 수 있습니다. 그래서 KDS는 “해석주기를 쓰더라도 너무 길면 \(C_u T_a\)로 제한하라”고 정하고 있습니다.

1-5. 비보강 채움벽이 있으면 왜 주기를 2/3로 줄일까?

Tₐ,infill = (2 / 3) · Cₜ · hₙˣ

비보강 채움벽(조적벽 등)이 들어가면, 건물은 더 이상 완전한 맨골조처럼 움직이지 않습니다. 지진 초기에 채움벽이 대각 압축 스트럿처럼 거동하여 횡강성을 키우기 때문입니다.

강성 K가 커지면 → 주기 T는 짧아진다

즉, 채움벽을 무시하면 맨골조보다 실제보다 더 “물렁한” 모델이 되어 주기가 길어지고, 그 결과 설계지진력이 작아질 위험이 있습니다.

맨골조 vs 채움벽 있는 골조

2. 모드 해석과 질량참여율의 중요성

u(t) = Σ ϕₘ qₘ(t)
u(t) : 구조물 전체 응답
ϕm : m차 모드형상
qm(t) : m차 모드의 시간응답

실제 지진응답은 1차모드가 끝나고 2차모드가 오는 식이 아니라, 여러 모드가 동시에 겹쳐서 나타납니다. 그래서 응답스펙트럼해석에서는 몇 개 모드까지 포함할지 결정하는 것이 중요합니다.

1차모드

전체가 한 번 크게 휘는 모양입니다.

2차모드

중간에 마디가 생기며 꺾이는 방향이 바뀝니다.

3차모드 이상

고층·비정형일수록 고차모드가 더 무시하기 어렵습니다.

구조기술사의 Tip · 줄넘기와 낚싯대

1차모드는 줄넘기를 한 번 크게 휘두를 때처럼 전체가 한 번 크게 움직이는 모양입니다. 고차모드는 낚싯대를 탁 쳤을 때처럼 중간중간 꺾이며 복잡하게 떠는 모양입니다. 고층으로 갈수록, 그리고 비정형이 커질수록 낚싯대 같은 고차모드의 존재감이 커집니다.

2-1. 질량참여율 90%가 왜 중요할까?

KDS는 응답스펙트럼해석에서 직교하는 각 방향에 대해 질량참여율 합계가 90% 이상이 되도록 모드 수를 정하라고 요구합니다. 한마디로 말하면, 실제 흔들리는 “몸집”을 충분히 담으라는 뜻입니다.

각 방향 질량참여율 합계 ≥ 90%
구조기술사의 Tip · 뷔페 접시

지진응답은 반찬을 하나씩 따로 먹는 게 아니라, 뷔페 접시에 여러 반찬을 한 번에 담는 것과 비슷합니다. 1차모드만 담으면 접시가 반쯤 빈 상태일 수 있습니다. 2차, 3차, 4차까지 더해 총량이 90% 이상이 되게 해야 “실제 응답의 한 끼”에 가까워집니다.

SRSS : R = √(Σ Rₘ²)
CQC : R = √(ΣΣ ρᵢⱼ RᵢRⱼ)

언제 잘 맞나 모드 주기 차이가 충분히 클 때 모드 주기 차이가 가까울 때

Cₘ = 0.85 · (V / Vₜ) ≥ 1.0

단단한 그릇 위에 젤리를 올려놓고 흔들면 젤리가 더 크게 출렁이는 걸 떠올리면 쉽습니다. 연약하고 깊은 지반일수록 지반 자체가 흔들림을 증폭할 수 있어서, 같은 건물이라도 더 큰 응답을 받을 수 있습니다.

3-2. 강기둥-약보(Strong Column – Weak Beam)

좋은 메커니즘

보가 먼저 항복하면 층 전체가 연성적으로 에너지를 흡수할 수 있습니다. 즉, 손상은 생겨도 예고 없이 전체가 무너지지 않도록 버티는 구조가 됩니다.

위험한 메커니즘

기둥이 먼저 약해지면 층붕괴 메커니즘으로 이어질 수 있습니다. 그래서 KDS는 가능한 한 강기둥-약보가 되도록 설계 철학을 잡습니다.

KDS 41은 보-기둥 연결부에서 가능한 한 강기둥-약보가 되도록 설계하고, 특수모멘트골조의 기둥은 강기둥-약보 조건을 만족하도록 요구합니다.
Part 2

실전 응답스펙트럼 작도 가이드편

이제는 “이론”을 “그림”으로 바꾸는 단계입니다. 아래 4단계 순서대로 따라가면 KDS 설계응답스펙트럼을 직접 그릴 수 있고, 최종적으로는 Cs와 V까지 연결할 수 있습니다.

1

지반과 지역 계수

Z, 위험도계수, 지반종류를 정하고 Fa·Fv를 구해 SDS, SD1을 계산합니다.

2

변곡점 계산

T0, TS, TL을 구하면 그래프 뼈대가 정해집니다. 어디서 꺾이는지가 보입니다.

3

구간별 식 적용

주기 구간에 따라 직선, 수평선, 1/T 곡선, 1/T² 곡선을 이어 작도합니다.

4

건물 주기 대입

우리 건물의 T를 대입해 Cs와 V를 산정하고, 설계지진력과 연결합니다.

Step 1. 지반 및 지역 계수 결정

1) 지역계수와 위험도계수

S = Z · I_H
S : 유효지반가속도
Z : 지진구역계수
IH : KDS 17의 위험도계수
지진구역 지진구역계수 Z KDS 41에서 자주 쓰는 IH = 2.0 적용 시 S

건축물용 KDS 41 응답스펙트럼에서는 보통 2400년 재현주기 위험도계수 2.0을 사용합니다.

2) Fa, Fv → SDS, SD1

S_DS = S · 2.5 · Fₐ · 2/3
S_D1 = S · Fᵥ · 2/3

SDS는 짧은 주기 쪽의 설계스펙트럼가속도, SD1은 1초 주기에서의 설계스펙트럼가속도입니다. 이 두 값이 정해지면 응답스펙트럼의 전체 형상이 사실상 결정됩니다.

Step 2. 설계응답스펙트럼 그래프의 변곡점 계산

T₀ = 0.2 · (S_D1 / S_DS)
T_S = S_D1 / S_DS
T_L = 5 s
통제주기 의미 그래프에서 보이는 변화
T0 초기 상승구간의 끝 짧은 주기 전이구간이 끝나고, 거의 일정한 가속도 구간으로 들어갑니다.
TS 가속도 plateau의 끝 여기서부터 1/T 형태가 시작되어 속도 민감 성격이 두드러집니다.
TL 장주기 전이점 여기서부터 1/T² 형태가 시작되어 변위 민감 성격이 강해집니다.

기억하기 쉽게 말하면, T0는 “그래프가 올라가다 평평해지는 곳”, TS는 “평평하던 그래프가 내려가기 시작하는 곳”, TL은 “내려가던 그래프가 더 가파르게 꺾이는 곳”입니다.

Step 3. 구간별 설계응답가속도 \(S_a(T)\)

0 ≤ T ≤ T₀ : Sₐ(T) = 0.4S_DS + 0.6S_DS (T / T₀)
T₀ < T ≤ T_S : Sₐ(T) = S_DS
T_S < T ≤ T_L : Sₐ(T) = S_D1 / T
T > T_L : Sₐ(T) = (S_D1 · T_L) / T²
작도용 점 주기 T 세로값 Sa 작도 메모
A 0 0.4 SDS 원점이 아니라 0.4SDS에서 시작
B T0 SDS 직선 상승구간의 끝
C TS SDS 수평 plateau의 끝
D TL SD1/TL 1/T 구간과 1/T² 구간의 연결점

Step 4. 그래프 해석 및 설계 적용

응답스펙트럼 그래프에서 우리 건물의 주기 \(T\)가 어디에 걸리는지 확인하면, 그 주기에서의 스펙트럼가속도 \(S_a(T)\)를 읽을 수 있습니다. 등가정적해석에서는 여기에 KDS의 지진응답계수 \(C_s\) 식을 연결해 밑면전단력 \(V\)를 구합니다.

V = Cₛ · W
기본식 : Cₛ = (I_E / R) · S_DS
상한식( T ≤ T_L ) : Cₛ ≤ (I_E / R)(S_D1 / T)
상한식( T > T_L ) : Cₛ ≤ (I_E / R)(S_D1·T_L / T²)
하한식 : Cₛ ≥ 0.044S_DS I_E ≥ 0.01
초보자용 한 줄 루틴

Z → S → Fa, Fv → SDS, SD1 → T0, TS, TL → 우리 건물의 T 대입 → Cs → V

실무에서 꼭 보는 점

수치해석 주기를 그대로 믿지 않고, 주기 상한값 CuTa와 채움벽 보정 여부까지 확인한 뒤 설계용 T를 정하는 습관이 중요합니다.
