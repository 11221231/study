변위응답스펙트럼 정리

먼저 딱 한 문장
실제로는 운동방정식을 각 주기 T마다 풀어서 응답스펙트럼을 얻고, 설계에서는 그 울퉁불퉁한 결과를 4개의 간단한 구간 식으로 정리해서 씁니다.
이 문서에서 약속하는 표기
Sa는 g 단위의 스펙트럼가속도, Sv는 의사속도(pseudo velocity), Sd는 스펙트럼변위로 설명합니다. 그래프에서는 Sv를 cm/s, Sd를 cm로 표시합니다.
1분 요약
1. 핵심 흐름: 어디서 시작해서 어디로 가는가?

복잡하게 보이지만, 실제 흐름은 아래 5단계입니다.

1단계 지반가속도 ag(t)를 준비합니다.
2단계 고유주기 T가 하나 정해진 1자유도 구조물에 넣고 운동방정식을 풉니다.
3단계 시간에 따른 상대변위 u(t)를 얻고, 가장 큰 값 max|u(t)|를 뽑습니다.
4단계 그것이 바로 그 주기에서의 Sd(T) 한 점입니다.
5단계 이 점을 모든 T에 대해 반복해 곡선을 만들고, 필요하면 Sv, Sa로 바꿉니다.

왜 T를 바꾸나?

짧은 건물, 중간 건물, 긴 건물은 지진에 다르게 반응합니다. 그래서 주기 T를 계속 바꿔가며 계산합니다.

왜 최대값만 뽑나?

설계에서는 “가장 위험했던 순간”이 중요하기 때문입니다. 그래서 시간 전체 중 최대응답을 사용합니다.

왜 Sa를 많이 쓰나?

설계하중을 잡을 때 가속도 값이 직접적으로 편리하기 때문입니다. 그래서 코드가 Sa 형태로 많이 정리됩니다.

왜 Sd도 중요하나?

건물이 실제로 얼마나 많이 밀리고 휘는지 보는 데에는 변위가 더 직접적이기 때문입니다.

한 줄 정리
운동방정식 → Sd를 먼저 얻고, 그다음 관계식으로 Sv, Sa를 연결하면 전체 그림이 잡힙니다.
용어 먼저
2. T0, TS, TL, SDS, SD1 뜻

이 다섯 개는 그래프 모양을 결정하는 핵심 버튼입니다.

SDS

짧은 주기 쪽 높이입니다. 아주 딱딱한 건물·낮은 건물 쪽에서 Sa 그래프의 기준 높이 역할을 합니다.

SD1

주기 1초에서의 설계 스펙트럼가속도입니다. 1/T 구간의 기준점 역할을 합니다.

TS

평평한 구간이 끝나는 주기입니다. 여기서부터 Sa가 1T 꼴로 줄어듭니다.

T0

처음 직선으로 올라가는 구간의 끝입니다. 이 뒤부터 Sa는 잠시 일정하게 유지됩니다.

TL

긴 주기 영역으로 넘어가는 경계입니다. 이 뒤부터는 변위가 거의 일정하도록 Sa가 더 빠르게, 즉 1T2 꼴로 감소합니다.

관계식

TS = SD1SDS T0 = 0.2TS

TL은 보통 기준서/지도에서 따로 주어집니다.

SDS, SD1는 어디서 오나요?

보통 지역의 지진위험도와 지반조건을 반영한 값 SMS, SM1를 먼저 구하고, 그중 23를 취해 설계용 값으로 씁니다.

SDS = 23 SMS SD1 = 23 SM1
헷갈리기 쉬운 점
TS와 T0는 SDS, SD1에서 계산할 수 있지만, TL은 보통 그렇게 자동으로 나오지 않습니다. TL은 기준이 따로 주는 값이라고 기억하면 됩니다.
출발점
3. 운동방정식과 1자유도 그림

실제 건물은 복잡하지만, 처음 이해할 때는 질량-스프링-댐퍼 1자유도로 보면 됩니다.

md2udt2 + cdudt + ku = -md2ugdt2 d2udt2 + 2ξωndudt + ωn2u = -d2ugdt2 ωn = √(km) Tn = 2πωn ξ = c2mωn
말로 해석하면
왼쪽은 구조물이 스스로 버티는 힘(질량·감쇠·강성)이고, 오른쪽은 지반이 흔들리면서 구조물을 강제로 흔드는 입력입니다.

왜 상대변위 u(t)를 쓰나요?

바닥도 같이 움직이기 때문입니다. “바닥에 비해 질량이 얼마나 늦게 따라오느냐”를 보면 내부변형을 잘 이해할 수 있습니다.

여기서 T는 누구의 주기인가요?

지진파의 주기가 아니라, 그 1자유도 구조물 자체의 고유주기입니다. 그래서 T를 바꾸면서 여러 구조물을 가정해 보는 것입니다.

응답스펙트럼의 시작점
어떤 T에 대해 이 운동방정식을 풀고 u(t)를 얻은 뒤, 그 최대값을 취하면 Sd(T) 한 점이 만들어집니다.
실제로 찍기
4. 운동방정식으로 응답스펙트럼 그래프를 어떻게 그리나?

아래 그림은 설명용 합성 지진파(synthetic wave)를 넣었을 때의 예시입니다. 계산 과정 자체는 “각 T마다 운동방정식을 풀고 최대값을 한 개 찍는 것”입니다.

입력 지반가속도 ag(t)
설명용 예시 지진가속도입니다. 세로축 단위는 g입니다.
예시 구조물의 상대변위 u(t) · T = 0.80s
이 그래프에서 가장 큰 절댓값이 바로 Sd(T = 0.80s) = max|u(t)| 입니다.

예시점의 Sd

9.913 cm

예시점의 Sv

77.859 cm/s

예시점의 Sa

0.6233 g

운동방정식 기반 Sa(T)
여기서는 예시 지진파로부터 얻은 의사가속도 스펙트럼입니다.
운동방정식 기반 Sv(T)
여기서는 설계에서 자주 쓰는 의사속도 스펙트럼으로 표시했습니다.
운동방정식 기반 Sd(T)
각 T마다 계산한 최대 상대변위의 집합입니다.
아주 중요
위의 그래프들은 한 개 예시 지진파를 직접 계산해서 얻은 실제 응답스펙트럼이라서 울퉁불퉁합니다. 설계기준의 4구간 그래프는 이와 같은 결과를 설계하기 쉽게 매끈하게 정리한 모델입니다.
Sa가 어디서 나오나?
5. Sd에서 Sv, Sa가 어떻게 유도되나?

이 부분이 핵심입니다. “변위를 알면 왜 속도와 가속도도 연결될까?”를 가장 쉬운 방법으로 보면, 한 주기에서의 응답을 사인파처럼 생각하면 됩니다.

① 변위

u(t) = U sin(ωnt)

한 주기에서 응답의 대표 크기를 U라고 두겠습니다.

② 속도

dudt = ωnU cos(ωnt)

사인파를 한 번 미분하면 cos가 되고, 앞에 ωn가 붙습니다.

③ 가속도

d2udt2 = -ωn2U sin(ωnt)

한 번 더 미분하면 ωn2가 붙습니다.

Sd = U Sv = ωnSd = 2πT Sd Sa(g) = ωn2Sdg = 4π2gT2 Sd
Sd = T2π Sv Sd = gT24π2 Sa(g) Sv = gT2π Sa(g)
왜 이게 중요한가?
설계기준은 보통 Sa로 식을 주지만, 실제로 구조물이 얼마나 움직이는지 보려면 Sd가 필요합니다. 위 식을 쓰면 한 그래프를 다른 그래프로 바꿀 수 있습니다.
“의사속도, 의사가속도”라는 말이 왜 나오나요?

엄밀하게는 최대 상대속도와 최대 상대가속도를 직접 계산할 수도 있습니다. 하지만 설계에서는 Sv = ωnSd, Sa = ωn2Sd 같은 관계를 많이 쓰기 때문에 이를 의사속도, 의사가속도라고 부릅니다. 초보 단계에서는 “Sd에서 연결되어 나오는 설계용 속도·가속도”라고 이해하면 충분합니다.

설계식 만들기
6. Sa(T) 4구간 식은 왜 이렇게 생기나?

여기서부터는 실제 울퉁불퉁한 스펙트럼을 설계용으로 정리한 4구간 모델을 설명합니다. 핵심은 “구간의 의미를 정하고, 경계에서 부드럽게 이어지도록 식을 만든다”입니다.

구간 Sa(T) 이 구간의 뜻
0 ≤ T < T0 Sa = SDS ( 0.4 + 0.6 TT0 ) 아주 짧은 주기. 시작점 0.4SDS에서 SDS까지 직선으로 연결합니다.
T0 ≤ T ≤ TS Sa = SDS 상수 가속도(등가속도) 구간으로 단순화합니다.
TS ≤ T ≤ TL Sa = SD1T 상수 속도 구간으로 보면 Sa는 1T 꼴이 됩니다.
TL ≤ T Sa = SD1TLT2 상수 변위 구간으로 보면 Sa는 1T2 꼴이 됩니다.

① 0 ~ T0: 왜 직선인가?

두 점 (0, 0.4SDS)와 (T0, SDS)를 가장 단순하게 잇는 식이 직선이기 때문입니다.

기울기 = SDS - 0.4SDST0 = 0.6SDST0 Sa = 0.4SDS + 0.6SDST0T = SDS(0.4 + 0.6TT0)

② T0 ~ TS: 왜 평평한가?

이 구간은 설계에서 상수 가속도로 이상화합니다. 그래서 가장 단순하게 높이를 일정하게 둡니다.

Sa = SDS

③ TS ~ TL: 왜 1/T 인가?

이 구간을 상수 속도 구간으로 보면 Sv = gT2πSa 가 거의 일정해야 합니다. 그러려면 Sa가 1T 꼴이어야 합니다.

Sv = const. ⟹ Sa = CT T = 1초에서 Sa = SD1 이도록 C = SD1 따라서 Sa = SD1T

④ TL ~ ∞: 왜 1/T² 인가?

이 구간은 상수 변위 구간으로 보는 이상화입니다. Sd = gT24π2Sa 이므로 Sd가 일정하려면 Sa는 1T2 꼴이어야 합니다.

Sd = const. ⟹ Sa = CT2 T = TL에서 앞 구간과 이어져야 하므로 CTL2 = SD1TL C = SD1TL 따라서 Sa = SD1TLT2
경계에서 정말 이어지는지 확인
T = T0이면 첫 식은 SDS가 되어 2구간과 이어지고,
T = TS이면 SD1TS = SDS가 되어 2구간과 3구간이 이어지고,
T = TL이면 SD1TLTL2 = SD1TL가 되어 3구간과 4구간이 이어집니다.
속도그래프 추가
7. 속도응답스펙트럼 Sv(T)은 어떻게 나오나?

위에서 본 관계식 Sv = gT2π Sa(g) 를 그대로 각 구간식에 곱하면 됩니다.

구간 Sv(T) 그래프 모양
0 ≤ T < T0 Sv = gSDS2π (0.4T + 0.6T2T0) T가 곱해져서 위로 휘는 곡선처럼 올라갑니다.
T0 ≤ T ≤ TS Sv = gSDS2πT 직선으로 증가합니다.
TS ≤ T ≤ TL Sv = gSD12π 상수, 즉 평평한 속도 구간입니다.
TL ≤ T Sv = gSD1TL2πT 1T 꼴로 감소합니다.

0 ~ T0

가속도식에 T가 한 번 더 곱해져서 천천히 위로 휘며 증가합니다.

T0 ~ TS

직선입니다. 상수가속도 구간이 속도에서는 선형 증가로 바뀝니다.

TS ~ TL

수평선입니다. 그래서 이 구간을 상수 속도 구간이라고 부릅니다.

TL ~ ∞

1/T 감소입니다. 긴 주기로 갈수록 속도도 서서히 줄어듭니다.

초보자용 기억법
Sa에 T를 한 번 곱하면 Sv가 됩니다. 그래서 그래프 모양도 “가속도보다 한 단계 덜 급한” 모양으로 바뀝니다.
변위그래프
8. 변위응답스펙트럼 Sd(T)은 어떻게 나오나?

이번에는 관계식 Sd = gT24π2Sa(g) 를 각 구간식에 곱하면 됩니다. 즉, T2가 붙는다는 것이 포인트입니다.

구간 Sd(T) 그래프 모양
0 ≤ T < T0 Sd = gSDS4π2 (0.4T2 + 0.6T3T0) 0에서 아주 부드럽게 시작하는 곡선입니다.
T0 ≤ T ≤ TS Sd = gSDS4π2T2 포물선처럼 빠르게 증가합니다.
TS ≤ T ≤ TL Sd = gSD14π2T 직선으로 증가합니다.
TL ≤ T Sd = gSD1TL4π2 상수, 즉 수평선이 됩니다.

0 ~ T0

아직 T가 아주 작아서 변위는 거의 0에서 시작합니다.

T0 ~ TS

Sa는 일정해도, T2 때문에 Sd는 빠르게 커집니다.

TS ~ TL

직선이 되므로, 주기가 길어질수록 변위가 거의 비례해서 커집니다.

TL ~ ∞

상수입니다. 아주 긴 주기에서는 변위가 거의 일정한 상한에 가까워진다고 보는 것입니다.

시험에서 자주 묻는 핵심 문장
상수 가속도 구간 T0 ~ TS 에서는 Sd ∝ T2, 상수 속도 구간 TS ~ TL 에서는 Sd ∝ T, 상수 변위 구간 T ≥ TL 에서는 Sd = const. 입니다.
구간 1 · 0 ~ T0

한 줄 느낌

아주 딱딱한 구조물 구간. 지반을 거의 따라가므로 상대변위는 아직 작습니다.

9. 0 ~ T0 구간

여기서는 Sa가 0.4SDS에서 SDS까지 직선으로 올라갑니다. 하지만 Sv에는 T가 한 번, Sd에는 T2가 붙기 때문에 그래프 모양이 달라집니다.

Sa

Sa = SDS(0.4 + 0.6TT0)

직선 상승

Sv

Sv = gSDS2π(0.4T + 0.6T2T0)

부드럽게 상승하는 곡선

Sd

Sd = gSDS4π2(0.4T2 + 0.6T3T0)

0에서 시작하는 완만한 곡선

왜 Sd는 0에서 시작하나요?
T가 0에 가까우면 T2와 T3도 0에 가까워집니다. 그래서 가속도 값이 0.4SDS에서 시작해도, 변위는 거의 0에서 출발합니다.
구간 2 · T0 ~ TS

한 줄 느낌

가속도는 일정, 속도는 선형 증가, 변위는 포물선처럼 빠르게 증가합니다.

10. T0 ~ TS 구간

이 구간은 상수 가속도 구간으로 단순화되어 있어서 가장 외우기 쉽습니다. Sa가 일정하기 때문에, 곱해지는 T와 T2의 영향이 더 뚜렷하게 보입니다.

Sa

Sa = SDS

수평선

Sv

Sv = gSDS2πT

직선

Sd

Sd = gSDS4π2T2

포물선

가장 많이 나오는 포인트
이 구간에서는 Sd ∝ T2 이므로, 주기가 조금만 길어져도 변위가 꽤 빨리 커집니다.

왜 TS = SD1/SDS 인가요?
경계 TS에서 2구간과 3구간이 이어져야 하므로 SDS = SD1TS 이어야 합니다. 이를 정리하면 TS = SD1SDS 가 됩니다.

14. 한 장으로 끝내는 최종 정리

시험 직전이나 과제 제출 전에 딱 이것만 다시 보면 됩니다.

출발

운동방정식 m u¨ + c u˙ + ku = -m u¨g 를 각 T마다 풀고, 최대 상대변위 max|u(t)|를 뽑으면 Sd(T) 한 점입니다.

연결식

Sv = 2πTSd, Sa(g) = 4π2gT2Sd

특성주기

TS = SD1SDS, T0 = 0.2TS, TL은 기준서/지도에서 줍니다.

구간별 모양

0~T0: 완만한 시작
T0~TS: Sd는 T2
TS~TL: Sv 일정, Sd는 T
TL~∞: Sd 일정

가장 중요한 한 줄
설계기준의 Sa 4구간 식에 gT2π를 곱하면 Sv, gT24π2를 곱하면 Sd가 됩니다.

1) 응답스펙트럼은 보통 상대응답 기준으로 이해합니다.
2) 설계 4구간 식은 실제 계산 결과를 다듬은 설계용 모델입니다.
3) 긴 주기 영역에서는 가속도가 작아져도 변위는 무시하면 안 됩니다.
